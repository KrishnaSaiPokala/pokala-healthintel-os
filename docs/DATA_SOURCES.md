# Data Sources

This project avoids broad scraping. It prioritizes official APIs, official bulk downloads, and narrow public machine-readable files.

Initial sources:

- NPPES NPI Registry API
- openFDA APIs and bulk downloads
- ClinicalTrials.gov API
- CMS Medicare utilization/payment public files
- CMS Open Payments public data
- CDC PLACES public data
- HHS OCR breach portal/public reports
- Hospital Price Transparency machine-readable files only after the MVP

## Rule

No PHI. No login walls. No permission-gated datasets. No paid APIs. No broad crawling.
